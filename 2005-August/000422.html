Looks like a server problem -- the lone dot on a line is supposed to 
indicate the end of the message, and if it appears in the message the 
server is supposed to make it two dots.  It's in the message just because 
of an unfortunate line break in its quoted-printable encoding.

Unfortunately there's not even any indication of what server software 
they're running.  The greeting flag is just:
+OK POP3 Ready poczta.o2.pl
